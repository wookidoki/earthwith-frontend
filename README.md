# EarthWith Frontend

환경 보호 커뮤니티 플랫폼 프론트엔드

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- Axios

## 주요 기능

- 로그인/회원가입
- 게시판
- 에코 피드
- 뉴스
- 탄소 계산기
- 랭킹 시스템
- 관리자 페이지

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
